# Graphql & Spring Boot 2
A demo project of graphql and spring boot 2.

##### Prerequitions:
- Java11
- Maven
- Node
 
Test and Build:<br />
`  mvn clean install
`

Run Application:<br />
`java -jar target/graphql-springboot-0.0.1-SNAPSHOT.jar
`


Once the app has started navigate to:<br />
`http://localhost:8080/graphiql
`
